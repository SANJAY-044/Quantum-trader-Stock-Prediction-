What This App Does Right Now

This app runs a real LSTM neural network entirely in your browser using TensorFlow.js. The stock price data is simulated using Geometric Brownian Motion — the same mathematical model used by real quant traders. The LSTM model truly trains; the loss curve, RMSE, and MAPE are real neural network outputs.

Option B — Replace Simulated Data (Code Changes)

Find the genData() function in the HTML source and replace it with this Alpha Vantage fetch:

// Replace genData(symbol) with this: async function fetchLiveData(symbol) { const KEY = 'YOUR_API_KEY_HERE'; const url = `https://www.alphavantage.co/query ?function=TIME_SERIES_DAILY &symbol=${symbol} &outputsize=full &apikey=${KEY}`; const res = await fetch(url); const json = await res.json(); const ts = json['Time Series (Daily)']; return Object.entries(ts) .map(([date, v]) => ({ date, open: +v['1. open'], high: +v['2. high'], low: +v['3. low'], close: +v['4. close'], vol: +v['5. volume'] })) .reverse(); // oldest → newest }

Tip: Cache the result in localStorage so you don't burn API quota on every page reload.

Option C — Full Python Backend (Production)

For a production system, run a lightweight Python server that fetches data and retrains the model server-side:

1 · Install Dependencies
# Install required packages pip install yfinance pandas numpy flask tensorflow scikit-learn
2 · Python Backend (server.py)
import yfinance as yf from flask import Flask, jsonify, request import numpy as np app = Flask(__name__) @app.route('/api/stock') def get_stock(): symbol = request.args.get('symbol', 'AAPL') df = yf.download(symbol, period='2y', interval='1d') df = df.reset_index() rows = [] for _, r in df.iterrows(): rows.append({ 'date': r['Date'].strftime('%Y-%m-%d'), 'open': round(float(r['Open']), 2), 'high': round(float(r['High']), 2), 'low': round(float(r['Low']), 2), 'close': round(float(r['Close']), 2), 'vol': int(r['Volume']) }) return jsonify(rows) if __name__ == '__main__': app.run(port=5000)
3 · Run the Server
python server.py # Server starts at http://localhost:5000 # Test: curl http://localhost:5000/api/stock?symbol=AAPL
4 · Connect This App to the Backend
// In the HTML, replace DB[s] = genData(s) with: async function loadStock(symbol) { const res = await fetch( `http://localhost:5000/api/stock?symbol=${symbol}` ); DB[symbol] = await res.json(); }
Auto-Retrain Schedule

For continuous real-time operation, set up automatic retraining when new market data arrives:

09:30 ET — Market Opens
Fetch overnight data from API. Load pre-trained model weights from localStorage or server. Show yesterday's predictions vs actual open prices.
Every 60 Minutes — Intraday Refresh
Fetch latest hourly candle. Append to window buffer. Re-run forward pass (predict only, no retrain) to update forecast in real time. Update BUY/SELL signal.
16:00 ET — Market Close
Append final closing price to dataset. Trigger full model retrain on updated dataset. Save new model weights. Log RMSE and MAPE to history.
After Hours — Evaluation
Compare today's LSTM forecast vs actual prices. Track prediction accuracy over rolling 30-day window. Alert if MAPE exceeds 5% threshold.
Hyperparameter Tuning Guide
ParameterRecommendedEffect
Window Size20–30 daysLarger = more context, slower training. 20 is standard for daily close.
Forecast Days5–15 daysBeyond 15 days accuracy drops sharply — compounding prediction errors.
Epochs60–100Watch the loss curve. Stop when it flattens (early stopping prevents overfit).
LSTM Units48–64More units = more capacity. Diminishing returns past 96 for daily stock data.
Learning Rate0.001Too high = diverges. Too low = slow. 0.001 is the Adam optimizer sweet spot.
Batch Size32Fixed in code. Larger = more stable gradients but more memory.
Deploy to the Web
1
GitHub Pages (Free, 5 min)
Push the HTML file to a GitHub repo → Settings → Pages → Deploy from main branch. Your app is live at username.github.io/repo. CORS headers may block direct API calls — use a proxy.
2
Netlify / Vercel (Free tier, with backend)
Deploy the HTML as a static site. Add a serverless function (/api/stock.js) that fetches from Alpha Vantage server-side, avoiding CORS. Deploy in under 2 minutes with drag-and-drop.
3
Docker + VPS ($5/mo)
Package the Python Flask server into a Docker container. Deploy to a DigitalOcean Droplet or AWS EC2. Set up a cron job for market-hours retraining. Use nginx as reverse proxy.
4
Streamlit Cloud (Free Python UI)
Rewrite the frontend in Streamlit for a fully Python solution. Use st.line_chart() for the price chart and st.cache_data for data caching. Deploy free at streamlit.io/cloud.
